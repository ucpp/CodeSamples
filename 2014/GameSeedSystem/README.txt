Система для генерации "сида" (строки-ключа) как, например,
в игре "The Binding of Isaac", по значению которого создается
генератор случайных чисел и вычисляется версия игры в которой
был создан этот "сид".

ChangeLog:

v.1.0.0
- генерация строки сида с сохранением в ней значения версии проекта
- возможность восстанавливать рандомайзер по ключу или по сиду
